== Installation ==

1. Upload `xpress.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

A video player can be embedded in a post using a the site shortcode 
with the ID of the video.

The id is the number associated with the videos on the respective site.
